# FlexBill - Flex Printing Billing Software

A full-stack Flex Printing Billing Software built using the MERN Stack.

## Features

- User Authentication (Login/Register)
- Dashboard Analytics
- Customer Management (CRUD)
- Product Management (CRUD)
- Payment Management
- Invoice Generation
- Invoice Printing
- Order Management
- Reports & Analytics
- Responsive Admin Dashboard

## Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM
- Recharts

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm run backend
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `.env` file in backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
